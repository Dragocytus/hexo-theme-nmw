https 无法访问：

云服务器请注意在控制台配置相关安全组并开启，http是 80 端口，https是 443 端口。

pm2启动失败：

非root用户下，即使执行sudo pm2 start app.js，也可能权限不够而启动失败。

请进入root账户，然后再执行pm2 start app.js来启动http和https服务器。
